Ajuda
Como obter a chave do CRM?
  1. Acesse o CRM DataCrazy
  2. Vá em Configurações
  3. Clique em Integrações → API
  4. Copie a chave gerada
Como obter o token de CPF?
  1. Acesse dash.cpf-brasil.org
  2. Crie uma conta ou faça login
  3. Vá em API → Token
  4. Copie seu token

Endpoints da API
Método Endpoint Descrição
GET /health Verifica se a API está online
POST /api/webhook/datacrazy Webhook principal do CRM DataCrazy
POST /api/consultar-cpf Consulta CPF diretamente
GET/POST /api/config Gerencia configurações da API
Exemplo de Requisição
curl -X POST https://sua-api.railway.app/api/webhook/datacrazy \
  -H "Content-Type: application/json" \
  -d '{
    "conversationId": "696e3ce76f116fa520361fe4",
    "leadPhone": "5511999999999",
    "mensagem": "Meu CPF é 123.456.789-00"
